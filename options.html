Jira Copy to Clipboard

Clipboard format

Placeholders

Click on a placeholder in the table below to insert it in the clipboard format input above.
You can also drag and drop the placeholders into the input field.

Placeholder Description
{key} Issue key (e.g., ABC-123)
{title} Issue title / summary
{type} Issue type (e.g., Bug, Task, Story)
{priority} Issue priority
{status} Issue status
{description} Issue description
{reporter} Issue reporter's display name
{assignee} Issue assignee's display name
{url} Issue URL
{linkStart} Start of issue hyperlink
{linkEnd} End of issue hyperlink

Examples

Example Result
{key}: {title} PROJ-123: My Jira issue
{linkStart}{key}{linkEnd}: {title} - {status} PROJ-123: My Jira issue - DONE

Tip

Copy your Jira info to the clipboard in text as well as HTML format
by using the tokens {linkStart} and {linkEnd}.
  • If you use paste into a plain text editor or an application that only supports plain text,
    you'll get a plain text version of the content without the hyperlink.
  • If you use paste into an HTML-aware application (like a rich text editor or an email client),
    youâ€™ll get the formatted HTML version with hyperlink to the issue. So you can quickly browse to it!
