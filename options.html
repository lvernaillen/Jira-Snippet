Click on a placeholder in the table below to insert it in the input field. You can also drag and drop the placeholders into the input field.

Placeholder Description
{key} Issue key (e.g., ABC-123)
{title} Issue title / summary
{description} Issue description
{type} Issue type (e.g., Bug, Task, Story)
{priority} Issue priority
{status} Issue status
{reporter} Issue reporter's display name
{assignee} Issue assignee's display name
{url} Issue URL
{linkStart} Start of issue hyperlink
{linkEnd} End of issue hyperlink

Examples:

Example Result
{key}: {title} PROJ-123: My first Jira issue
{linkStart}{key}{linkEnd}: {title} ({status}) PROJ-123: My first Jira issue (DONE)

Tip:

Why not add a hyperlink using "{linkStart}" and "{linkEnd}" by default?
It will copy your Jira info to the clipboard in text as well as HTML format.
  • If you use paste into a plain text editor or an application that only supports plain text, you'll get a plain text version of the content without the hyperlink.
